### Demo

To test out the demo, head to [demo.polr.me](http://demo.polr.me) and use the following credentials:

- Username: `demo-admin`
- Password: `demo-admin`

### 升级

 - 备份
 - 使用 `git pull` 
 - 运行 `composer install --no-dev -o` 
 - 运行 `php artisan migrate` 去确认数据库升级
